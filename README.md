# ⚡ Energy Consumption Sustainability Analysis

An end-to-end Data Analytics project focused on analyzing building energy consumption, greenhouse gas emissions, and sustainability metrics using Python, SQL, and Power BI.

---

## 📌 Project Overview

This project analyzes energy consumption data from buildings to identify usage patterns, compare property performance, and uncover opportunities to improve energy efficiency.

The project demonstrates the complete data analytics workflow-from data cleaning and exploratory data analysis to SQL querying and interactive dashboard creation.

---

## 🎯 Objectives

- Clean and preprocess raw energy consumption data
- Perform exploratory data analysis (EDA)
- Analyze energy usage using SQL
- Build an interactive Power BI dashboard
- Generate meaningful sustainability insights
- Support data-driven decision making

---

# 📈 Key Insights

- Energy consumption varies significantly across different property types.
- Buildings with higher energy usage generally produce higher greenhouse gas emissions.
- Older buildings often show lower energy efficiency.
- Certain building categories consistently consume more electricity than others.
- Interactive dashboards help identify energy-intensive properties quickly.
